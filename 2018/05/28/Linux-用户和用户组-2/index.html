创建用户


  • UID 上一个新建用户UID+1

  • 家目录 /home/用户名

  • 用户备注 无

  • 主用户组 新建与用户同名的用户组

  • 从属用户组

  • Shell类型 bash

useradd -c clan -u 601 -d /tmp/stu01 clannad -G stu.s_group01

  • -c指定用户备注

  • -u指定用户UID

  • -d指定用户家目录

  • -g指定主用户组

  • -G指定除主用户组外所从属的用户组


    • 可用finger和id来检验
