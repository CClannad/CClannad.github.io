Linux 重定向（Redirection）和管道（Pipes）

发表于 2018-05-25

echo

echo 用来创建空文本
touch 用来创建空文件

>输出重定向

1> stdout.txt 将正常输出输出到对应文件
2> stderr.txt 将错误输出输出到对应文件
2>>/dev/null 将错误输出输出到空文件
/dev/null 视为黑洞，写入的文件将被丢弃

如果存在对应文件单个>将会覆盖，否则就创建新文件，如果不覆盖，那么用>>将新内容写到文件元内容的下方。
&> std.txt可以 用来讲错误输出和正常输出输出到一个文本中

\< <<输入重定向

tr

tr [ ] [ ] <[文本]

例如 tr ‘ ‘ ‘\t’ <./fruit.txt> 将当前目录中，fruit文本中的空格替换为’\t’

sort

sort -k <<EOF

> 1 apple

> 2 pear

> 3 banana

例如 sort -k2 <<EOF 表示按第二个字段排序，界定符为EOF
也可以将排序后的内容输入到对应文本，可以在后面自己写
